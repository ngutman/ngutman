# Hi, I'm Nimrod "Guti" Gutman

📍 **Tel Aviv** | **Backend Engineering** | 🤖 **AI Workflows**

I have been taking computers apart since I was a kid messing with a cranky old i386 in the early 90s, Playing golden-axe and Commander Keen. I turned that curiosity and passion into a software career, and I still learn the same way: build something real, break it, understand it and try to rebuild it better. These days I spend most of my time around AI tooling, agentic workflows, home automation, keyboards, and projects where software gets to touch the physical world.
